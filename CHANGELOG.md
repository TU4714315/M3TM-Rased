# 📋 سجل التغييرات - Changelog

## v4.0.0 - 2026-05-17 🎨
### ✨ التحسينات
- **Dark/Light Mode**: تبديل سلس بين المظاهر الفاتحة والداكنة
- **بحث متقدم**: ابحث عن الأخبار حسب العنوان أو المصدر أو التصنيف
- **فلترة ذكية**: فلتر متعدد (AND logic) - التصنيف + الأهمية + البحث
- **عداد فوري**: شاهد عدد الأخبار المفلترة أثناء البحث
- **إضافة مصادر محسّنة**: اضف اسم + رابط + وصف لكل مصدر
- **بحث في المصادر**: جد أي مصدر بسرعة
- **ترقية الأمان**: PIN من 4 إلى 6 أرقام
- **ألوان ديناميكية**: واجهة تتكيف مع المظهر المختار
- **أيقونات محسّنة**: علامات ملونة تميز درجات الأهمية
- **واجهة مستجيبة**: تصميم أفضل على جميع الأجهزة

### 🐛 إصلاحات
- تحسين استجابة الواجهة على الشاشات الصغيرة
- إضافة حفظ تلقائي لاختيار المظهر

### 📝 ملاحظات تقنية
- تغيير مفتاح التخزين من `v2` إلى `v3` (بيانات جديدة)
- إضافة `THEME_KEY` لحفظ تفضيل المظهر
- PIN أصبحت 6 أرقام بدلاً من 4

## ملاحظات مهمة 📌

### تحديث البيانات
عند الترقية من إصدار سابق:
```javascript
// البيانات القديمة (v2) في localStorage
m3tm_rased_data_v2

// البيانات الجديدة (v3) في localStorage
m3tm_rased_data_v3

// يمكنك نقل البيانات يدوياً إذا أردت
```

### تفضيل المظهر
يتم حفظ تفضيل Dark/Light Mode في:
```javascript
localStorage.getItem("m3tm_theme")  // "dark" أو "light"
```
