## 💡 نصائح مهمة

### ✅ للحصول على أفضل تجربة
```
1. استخدم متصفح حديث (Chrome/Firefox/Safari)
2. فعّل JavaScript في المتصفح
3. لا تستخدم Private/Incognito Mode
4. امسح الذاكرة المؤقتة إذا واجهت مشاكل
5. تأكد من توفر مساحة تخزين كافية
```

### ⚠️ أشياء يجب تجنبها
```
1. لا تحذف البيانات من localStorage يدوياً
2. لا تشارك جهازك مع آخرين
3. لا تستخدم نفس PIN لعدة حسابات
4. لا تعتمد على الموقع كـ Backup فقط
5. لا تثق ببيانات الـ Demo بشكل كامل
```
