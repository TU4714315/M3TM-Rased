## 📋 أنواع المساهمات

### 🐛 الإبلاغ عن الأخطاء
1. تحقق من أن الخطأ لم يتم الإبلاغ عنه من قبل
2. افتح Issue جديدة
3. اشرح:
   - ما تتوقعه
   - ما يحدث بالفعل
   - خطوات إعادة الإنتاج
   - المتصفح والجهاز

### 💡 اقتراح تحسينات
1. فتح Issue للنقاش أولاً
2. اشرح الفائدة والحالات الاستخدام
3. قدم أمثلة إذا أمكن

### 🎨 تحسينات الواجهة
- تصاميم أفضل
- تحسينات الاستجابة
- أيقونات وألوان

### 🔒 تحسينات الأمان
- إغلاق الثغرات الأمنية
- تحسينات التشفير
- معالجة أفضل للأخطاء

## 🛠️ عملية العمل

### 1. انسخ المستودع
```bash
git clone https://github.com/YOUR_USERNAME/M3TM-Rased.git
cd M3TM-Rased
```

### 2. انشئ فرع عمل
```bash
git checkout -b feature/your-feature-name
```

### 3. قم بالتغييرات
- عدّل الملفات المطلوبة
- اختبر على متصفحات مختلفة
- تأكد من التوافقية مع الهاتف

### 4. الالتزام
```bash
git commit -m "feat: إضافة ميزة جديدة"
git commit -m "fix: إصلاح خطأ"
```

### 5. أرسل PR
```bash
git push origin feature/your-feature-name
```
