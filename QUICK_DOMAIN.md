## 📋 الخطوات بالترتيب

### الخطوة 1: تسجيل على Freenom ⏱️ 5 دقائق

```
1. اذهب إلى: https://www.freenom.com
2. اضغط: Sign Up
3. أملأ البريد + كلمة المرور
4. تحقق من بريدك وفعّل الحساب
```

### الخطوة 2: شراء الدومين مجاناً ⏱️ 5 دقائق

```
1. ابحث عن: m3tm-rased
2. اختر: .ml (مجاني)
3. أضف للسلة
4. اختر: 12 months
5. أكمل الشراء (بدون دفع)
```

### الخطوة 3: إعداد Cloudflare ⏱️ 10 دقائق

```
1. اذهب إلى: https://www.cloudflare.com
2. اضغط: Sign Up
3. أضف الموقع (Add a Site)
4. أدخل: m3tm-rased.ml
5. انسخ Nameservers (ستظهر 4 قيم)
```

### الخطوة 4: تحديث Nameservers في Freenom ⏱️ 2 دقيقة

```
1. رجع إلى: Freenom.com
2. My Domains → m3tm-rased.ml
3. Manage Domain → Management Tools
4. Nameservers → Use Custom Nameservers
5. أضف الـ 4 Nameservers من Cloudflare
6. احفظ
```

### الخطوة 5: إضافة CNAME في Cloudflare ⏱️ 1 دقيقة

```
1. في Cloudflare DNS
2. اضغط: Add Record
3. Type: CNAME
4. Name: @ (جذر الدومين)
5. Target: tu4714315.github.io
6. Proxy Status: Proxied (زرقاء)
7. احفظ
```

### الخطوة 6: تفعيل في GitHub ⏱️ 1 دقيقة

```
1. اذهب إلى: GitHub Settings → Pages
2. Custom domain: m3tm-rased.ml
3. احفظ
4. انتظر 5 دقائق
5. تفعيل: Enforce HTTPS
```

⏳ **الانتظار**: 15-30 دقيقة للانتشار
