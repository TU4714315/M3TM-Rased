# 🔐 M3TM RASED - نظام الرصد الأمن

نظام رصد إخباري احترافي مع نظام PIN آمن بدون الحاجة لـ backend

## ✨ المميزات

### 🔐 الأمان
- **نظام PIN محلي**: تسجيل دخول آمن بـ 4 أرقام
- **تشفير البيانات**: جميع البيانات مشفرة في localStorage
- **بدون backend**: لا توجد خوادم خارجية - كل شيء محلي
- **جلسات آمنة**: معرّفات مشفرة في المتصفح

### 👥 إدارة المستخدمين
- **أدوار مختلفة**: مسؤول، محرر، مستخدم
- **إضافة مستخدمين ديناميكياً**: إنشاء PINs جديدة بدون تعديل الكود
- **إدارة كاملة**: حذف المستخدمين بسهولة
- **تتبع المستخدمين**: معرفة من أضاف الخبر أو المصدر

### 📰 إدارة الأخبار
- **صفحة عامة**: عرض الأخبار للجميع (قراءة فقط)
- **لوحة إدارية**: إضافة/حذف الأخبار (للمسؤولين فقط)
- **تصنيفات متعددة**: الحج، المدينة، باكستان، OSINT، إلخ
- **درجات أهمية**: حرج، عالي، متوسط، منخفض
- **روابط وملاحظات**: معلومات إضافية لكل خبر

### 📚 مكتبة المصادر
- **تنظيم المصادر**: إضافة روابط المواقع والصحف
- **إدارة سهلة**: حذف المصادر غير المستخدمة
- **تتبع التواريخ**: معرفة متى تمت إضافة كل مصدر

### 🌍 World Monitor
- **تضمين مواقع خارجية**: تحميل مراقبة العالم مباشرة
- **وضع ملء الشاشة**: عرض بحجم أكبر للتركيز
- **فتح خارجي**: فتح في نافذة جديدة

## 🚀 البدء السريع

### الوصول للنظام

1. **افتح الصفحة**: انقر على `index.html` أو انسخ الرابط
2. **شاشة تسجيل الدخول**:
   - أدخل 4 أرقام (رقم PIN)
   - PINs الافتراضية:
     - **مسؤول**: `0000`
     - **مستخدم**: `1234`

### واجهة المستخدم

#### للمستخدمين العاديين:
- 📊 **لوحة الرصد**: إحصائيات سريعة
- 📰 **الأخبار**: عرض فقط (لا يمكن التعديل)
- 📚 **المصادر**: عرض المصادر المتاحة
- 🌍 **World Monitor**: شاشة المراقبة

#### للمسؤولين:
- جميع ما للمستخدمين + الصلاحيات التالية:
- ⚙️ **الإدارة**: إضافة/حذف الأخبار
- 👥 **المستخدمون**: إنشاء/حذف المستخدمين الجدد

## 🌐 نشر على GitHub Pages

### الخطوات:

1. اذهب إلى `Settings` > `Pages`
2. اختر `Deploy from a branch`
3. اختر `main` و `/root`
4. ستحصل على رابط: `https://yourusername.github.io/M3TM-Rased`

### ربط دومين مخصص (Freenom + Cloudflare)

**راجع SETUP_GUIDE.md للتفاصيل الكاملة**

## 🐛 استكشاف الأخطاء

**المشكلة**: لا تعمل البيانات المحفوظة
- تأكد من تفعيل localStorage
- امسح ملفات التخزين المؤقت
- جرب متصفح مختلف

**المشكلة**: تسجيل الدخول لا يعمل
- تأكد من إدخال 4 أرقام فقط
- افتح أدوات المطور (F12)
