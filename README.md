# 🔐 M3TM RASED 2.0 - نظام الرصد الآمن

نظام رصد احترافي لإدارة الأخبار والمستخدمين مع نظام PIN آمن | Secure News Management System

## ✨ المميزات الرئيسية

### 🔐 الأمان والخصوصية
- **نظام PIN محلي**: فصل كامل بين المستخدمين والمسؤولين (PIN `0000` للإدارة، `1234` للمستخدمين)
- **تخزين محلي 100%**: جميع البيانات في localStorage - لا تغادر المتصفح
- **بدون Backend**: تطبيق ويب ثابت (Static) بملف HTML واحد فقط
- **جلسات آمنة**: SessionStorage للتحقق المؤقت

### 👥 إدارة المستخدمين الذكية
- **دورين متكاملين**: مشاهد (عام) و مسؤول (إدارة)
- **إنشاء مستخدمين ديناميكياً**: أضف PINs جديدة برقم إداري
- **إدارة كاملة**: حذف المستخدمين بسهولة
- **تسجيل دخول آمن**: 4 أرقام محلية فقط

### 📰 إدارة الأخبار الاحترافية
- **واجهة عامة**: عرض الأخبار للجميع (قراءة فقط)
- **لوحة إدارية**: إضافة/تعديل/حذف الأخبار (للمسؤولين)
- **تصنيفات متقدمة**: الحج، المدينة المنورة، باكستان، إيران، أمني، OSINT، عام
- **درجات أهمية**: عالي جداً، مهم، عادي
- **روابط ومعلومات**: مصدر، رابط خارجي، ملاحظات إضافية، تاريخ الإضافة

### 📚 مكتبة المصادر المتقدمة
- **تنظيم الروابط**: إضافة/حذف مصادر إعلامية
- **تتبع التواريخ**: معرفة متى تمت إضافة كل مصدر
- **تصفية سهلة**: بحث وتصنيف المصادر

## 🚀 البدء الفوري

### 1️⃣ الوصول للموقع
```
https://tu4714315.github.io/M3TM-Rased/
```

### 2️⃣ تسجيل الدخول

| PIN | الدور | الوصول |
|-----|------|--------|
| `0000` | مسؤول | جميع الصلاحيات |
| `1234` | مستخدم | عرض فقط |

### 3️⃣ إنشاء مستخدم جديد
1. اضغط **"مستخدم جديد"** في صفحة التسجيل
2. أدخل PIN الإدارة: `0000`
3. أنشئ PIN وافم جديد
4. اضغط **"إضافة"**

```

```


## 🛡️ معايير الأمان

### ✅ ما هو آمن
- ✅ لا توجد طلبات API خارجية
- ✅ البيانات لا تغادر المتصفح الخاص بك
- ✅ بدون خوادم أو قواعد بيانات
- ✅ بدون معلومات حساسة محفوظة

### ⚠️ تحذيرات
- ⚠️ الأمان محلي فقط (للاستخدام الشخصي/الفريق)
- ⚠️ إذا حذف المستخدم localStorage تُفقد جميع البيانات
- ⚠️ للأمان الإنتاجي الكامل، نحتاج Backend مستقبلاً
