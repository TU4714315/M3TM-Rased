# 🔐 M3TM RASED 3.0 - نظام الرصد الآمن المتقدم

نظام رصد احترافي لإدارة الأخبار والمستخدمين مع نظام PIN آمن | Advanced Secure News Management System

## ✨ المميزات الرئيسية

### 🔐 الأمان والخصوصية
- **نظام PIN محلي** 6 أرقام: فصل كامل بين المستخدمين والمسؤولين
- **تخزين محلي 100%**: جميع البيانات في localStorage - لا تغادر المتصفح
- **بدون Backend**: تطبيق ويب ثابت (Static) بملف HTML واحد فقط (~27 KB)
- **جلسات آمنة**: SessionStorage للتحقق المؤقت

### 👥 إدارة المستخدمين الذكية
- **دورين متكاملين**: مشاهد (عام) و مسؤول (إدارة)
- **إنشاء مستخدمين ديناميكياً**: أضف PINs جديدة برقم إداري (`123987`)
- **إدارة كاملة**: حذف المستخدمين بسهولة
- **تسجيل دخول آمن**: 6 أرقام محلية بدون تشفير خارجي

### 📰 إدارة الأخبار الاحترافية
- **واجهة عامة**: عرض الأخبار للجميع (قراءة فقط)
- **لوحة إدارية**: إضافة/تعديل/حذف الأخبار (للمسؤولين)
- **تصنيفات متقدمة**: الحج، المدينة، باكستان، إيران، أمني، OSINT، عاجل، عام
- **درجات أهمية**: عالي جداً ⚡، مهم ⭐، عادي 📌
- **روابط ومعلومات**: مصدر، رابط خارجي، ملاحظات إضافية، تاريخ الإضافة
- **🔍 بحث متقدم**: ابحث عن الأخبار حسب العنوان أو المصدر
- **🎨 فلترة ذكية**: فلّتر حسب التصنيف والأهمية

### 📚 مكتبة المصادر الكاملة
- **إضافة مصادر**: أضف روابط الصحف والمواقع الإعلامية
- **وصف المصدر**: أضف وصف لكل مصدر
- **حذف سهل**: احذف المصادر غير المستخدمة
- **بحث سريع**: ابحث عن أي مصدر
- **تتبع التواريخ**: معرفة متى تمت إضافة كل مصدر

```
3. الرابط الافتراضي: https://tu4714315.github.io/M3TM-Rased
```

### 🌍 إضافة دومين مخصص (.ml مجاني)

#### أ) تسجيل الدومين
```
Freenom.com → اختر .ml أو .tk → سجّل مجاناً
```

#### ب) ربط مع Cloudflare
```
Cloudflare.com → أضف الموقع → اختر nameservers
```

#### ج) في Cloudflare DNS - أنشئ CNAME
```
Type:  CNAME
Name:  @ (أو اسم الدومين)
Value: tu4714315.github.io
TTL:   Auto
```

#### د) في GitHub Pages
```
Settings → Pages → Custom domain: yourdomain.ml
```

**النتيجة**: 🎉 موقعك يعمل على `https://yourdomain.ml` مع HTTPS مجاني
