# قوائم المراقبة والتنبيهات

## أنواع القوائم

دولة، شركة، جهة حكومية، شخص، دومين، بريد، CVE، جماعة/فصيل، كلمة مفتاحية،
مستودع GitHub، ومصدر إخباري.

تحتوي القائمة كلمات وكيانات وقنوات إخطار. القناة الفعلية الحالية هي لوحة التحكم.
حقلا البريد وTelegram جاهزان للتوصيل بمزود خارجي دون وضع أسراره في الواجهة.

## المطابقة

بعد حفظ خبر أو مؤشر أو مستودع، يفحص العامل القوائم المفعلة. عند التطابق:

1. ينشئ `watchlist_hits`.
2. يحفظ سبب التطابق والكلمات والدرجة.
3. ينشئ `alerts` عند الحاجة.

مستويات العرض: معلومات، تنبيه، عالي، حرج. تنبيهات الفشل المتكرر والمخاطر الحرجة
تظهر في `#/alerts`.

## الصلاحيات

المستخدم يرى ويدير قوائمه فقط. Manager وAdmin يستطيعان إدارة المحتوى، بينما
الكتابة الآلية للتطابقات والتنبيهات محصورة في Admin SDK.
