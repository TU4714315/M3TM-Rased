# التقارير التنفيذية

المسار `#/reports` يعرض تقارير `intelligence_reports`. زر
`إنشاء تقرير تنفيذي` يبني موجزًا عربيًا من أحدث الأخبار والمؤشرات العامة
والمستودعات المحفوظة.

أنواع التقارير المدعومة:

- موجز استخباري إقليمي
- تقرير التسريبات والمصادر الرمادية
- تقرير الخليج وإيران
- تقرير التجسس والاستخبارات
- تقرير الضربات والهجمات
- تقرير المستودعات والأدوات الجديدة
- تقرير مراقبة قائمة محددة
- تقرير أسبوعي تنفيذي

المسارات:

```text
POST /reports/intelligence
POST /reports/arabic-brief
POST /reports/grey-intel
GET  /reports/{id}
```

كل تقرير يتضمن نطاق التغطية، الملخص التنفيذي، المؤشرات، المخاطر، التوصيات،
المصادر، والتنبيه القانوني. التقرير تحليلي ويحتاج تحققًا بشريًا قبل توزيعه.
