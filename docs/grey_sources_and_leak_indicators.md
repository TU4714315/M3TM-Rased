# المصادر الرمادية ومؤشرات التسريب

## السياسة

الوحدة مخصصة للمصادر العامة والأبحاث الأمنية والنشرات الرسمية فقط. لا تنفذ:

- وصولًا مباشرًا للشبكة المظلمة.
- تجاوز تسجيل الدخول أو الدعوات أو CAPTCHA.
- تنزيل قواعد بيانات مسربة أو dumps.
- جمع كلمات مرور أو رموز وصول أو بيانات شخصية مسروقة.
- كشط مجموعات Telegram خاصة أو منتديات مدفوعة.

كل عنصر يعرض وسم `مؤشرات فقط` والتنبيه:

> هذا التقرير يعتمد على مصادر عامة ومؤشرات منشورة. لا يحتوي التقرير على بيانات مسربة خام أو كلمات مرور أو معلومات شخصية حساسة.

## التخزين

تخزن `grey_intel_items` العنوان والرابط والمصدر والتاريخ والتصنيف والجهات المتأثرة
والخطر والملخص القانوني. الحقل `rawPayloadMetadataOnly` يحتوي مراجع مزود محدودة،
ولا يحتوي جسم الاستجابة.

عندما تكون `data_sensitivity` مساوية لـ`محظور التخزين` يصبح
`contentSnippet_ar` فارغًا ولا يحفظ المحتوى.

القيم المدعومة:

- `عام`
- `حساس`
- `مؤشر تسريب`
- `تسريب محتمل`
- `محظور التخزين`

## المصادر

تشمل البذور CISA KEV ومدونات الأمان العامة لمزودين وباحثين ووسائل أمنية معروفة.
كل مصدر قابل للتعطيل أو الحذف من الإدارة. فشل RSS يظهر في سجل الجلب بوضوح.
