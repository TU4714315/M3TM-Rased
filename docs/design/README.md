# Design Reference

Visual reference:

- `docs/design/m3tm-raseed-command-center-reference.svg`

Product identity:

- `M3TM.RASEED`
- `مركز الرصد العربي`
- `لوحة الرصد الاستخباري`
- `رصد. تحليل. تنبيه. تقرير.`

The dashboard direction is a dark Arabic RTL command center with right navigation, compact KPI strip, risk map, timeline chart, event feed, gray sources, repository intelligence, and system health.

Recommended UI refinements:

- keep the same layout family
- improve spacing and typography
- improve icon consistency
- improve risk badge readability
- improve desktop, tablet, and mobile behavior
- avoid horizontal overflow on mobile
