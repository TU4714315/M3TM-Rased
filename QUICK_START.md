## ⚙️ ماذا يمكنك أن تفعل؟

### 👨‍💼 المسؤول (PIN: 123987)

#### ➕ إضافة خبر جديد:
```
1. اضغط "إضافة خبر" (من الأزرار العلوية)
2. أملأ البيانات:
   - العنوان: (مثل "تطورات في المنطقة")
   - المصدر: (مثل "BBC عربي")
   - الرابط: (اختياري - رابط الخبر)
   - التصنيف: (اختر من القائمة)
   - الأهمية: (اختر عادي/مهم/عالي جداً)
   - الملاحظات: (اختياري - معلومات إضافية)
3. اضغط "حفظ الخبر"
```

#### 📚 إضافة مصدر:
```
1. اذهب إلى "المصادر"
2. اضغط "إضافة مصدر"
3. أملأ:
   - الاسم: (مثل "BBC عربي")
   - الرابط: (مثل https://bbc.com)
   - الوصف: (اختياري)
4. اضغط "حفظ المصدر"
```

#### 👥 إضافة مستخدم:
```
1. اذهب إلى "المستخدمين"
2. أدخل:
   - PIN جديد: (مثل 5678)
   - الاسم: (مثل "محمد")
3. اضغط "إضافة مستخدم"
```

### 👤 المستخدم العام (PIN: 1234)

#### 📖 قراءة الأخبار:
```
1. اذهب إلى "الأخبار"
2. شاهد جميع الأخبار مع البيانات الكاملة
3. اضغط على الروابط الزرقاء لفتح الخبر
```

#### 🔍 البحث والفلترة:
```
1. في "الأخبار" ستجد:
   - 🔍 حقل البحث (ابحث عن كلمة)
   - 📂 فلتر التصنيف
   - ⭐ فلتر الأهمية
2. اكتب/اختر وستظهر النتائج فوراً
```

#### 📚 قراءة المصادر:
```
1. اذهب إلى "المصادر"
2. شاهد جميع المصادر المضافة
3. اضغط على الروابط الزرقاء لفتح المصدر
```

```
من صفحة التسجيل:
1. اضغط "إعداد رقم سري جديد"
2. أدخل PIN الإدارة: 123987
3. أدخل PIN الجديد: (أي رقم)
4. أدخل الاسم
5. اضغط "إضافة مستخدم"
```

```
اضغط "تسجيل خروج" (الزر الأحمر) من الأعلى
سيعود لشاشة التسجيل
```
