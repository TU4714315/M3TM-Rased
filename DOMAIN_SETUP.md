## 🎯 خطة التنفيذ الكاملة

### المرحلة 1: تسجيل الدومين على Freenom ⏱️ (10 دقائق)

#### الخطوة 1: الدخول إلى Freenom
```
1. اذهب إلى: https://www.freenom.com
2. اضغط "Register Now" أو "Sign Up"
3. أملأ البيانات:
   - البريد الإلكتروني
   - اسم المستخدم
   - كلمة المرور
4. تحقق من بريدك الإلكتروني
5. تسجيل الدخول
```

#### الخطوة 2: البحث عن الدومين
```
1. في الصفحة الرئيسية، اكتب: m3tm-rased
2. اختر الإضافة: .ml (أو .tk أو .ga إن لم توجد)
3. اضغط "Check Availability"
4. اختر المدة: 12 months (مجاني)
5. اضغط "Checkout"
```

#### الخطوة 3: إكمال التسجيل
```
1. في السلة، اضغط "Checkout"
2. أكمل البيانات الشخصية
3. أقبل الشروط
4. اختم التسجيل (بدون دفع = مجاني)
5. ستحصل على تأكيد البريد
```

✅ **النتيجة**: سيكون لديك دومين: `m3tm-rased.ml`

---

### المرحلة 2: ربط مع Cloudflare ⏱️ (5 دقائق)

#### الخطوة 1: التسجيل في Cloudflare
```
1. اذهب إلى: https://www.cloudflare.com
2. اضغط "Sign Up"
3. أدخل بريدك الإلكتروني
4. أدخل كلمة المرور
5. تحقق من بريدك
```

#### الخطوة 2: إضافة الموقع
```
1. اضغط "Add a Site"
2. أدخل: m3tm-rased.ml
3. اختر الخطة المجانية
4. اضغط "Continue"
```

#### الخطوة 3: نسخ Nameservers
```
سترى الـ Nameservers التالية (نسخها):
- ns1.cloudflare.com
- ns2.cloudflare.com
- ns3.cloudflare.com
- ns4.cloudflare.com

احفظها! ستحتاجها الآن
```

---

### المرحلة 3: تحديث Nameservers في Freenom ⏱️ (5 دقائق)

#### الخطوات:
```
1. اذهب إلى: https://www.freenom.com
2. تسجيل دخول بحسابك
3. اذهب إلى: "My Domains"
4. اختر: m3tm-rased.ml
5. اضغط: "Manage Domain"
6. اختر: "Management Tools"
7. اختر: "Nameservers"
8. اختر: "Use Custom Nameservers"
9. أضف:
   - ns1.cloudflare.com
   - ns2.cloudflare.com
   - ns3.cloudflare.com
   - ns4.cloudflare.com
10. احفظ التغييرات
```

✅ **النتيجة**: الـ DNS تشير الآن إلى Cloudflare

---

### المرحلة 4: إضافة CNAME في Cloudflare ⏱️ (2 دقيقة)

#### الخطوات:
```
1. اذهب إلى Cloudflare Dashboard
2. اختر الموقع: m3tm-rased.ml
3. اذهب إلى: DNS
4. اضغط: "Add Record"
5. اختر Type: CNAME
6. أملأ:
   - Name: @ (الجذر)
   - Target: tu4714315.github.io
   - TTL: Auto
   - Proxy Status: Proxied (الزرقاء)
7. اضغط: Save
```

✅ **النتيجة**: CNAME موجهة إلى GitHub Pages

---

### المرحلة 5: تفعيل Custom Domain في GitHub ⏱️ (1 دقيقة)

#### الخطوات:
```
1. اذهب إلى: https://github.com/TU4714315/M3TM-Rased
2. اختر: Settings
3. اختر: Pages (من القائمة اليسرى)
4. في "Custom domain" أدخل: m3tm-rased.ml
5. اضغط: Save
6. انتظر 5 دقائق لتفعيل SSL
7. تفعيل: "Enforce HTTPS"
```

✅ **النتيجة**: سيظهر ✅ أخضر عند التفعيل
